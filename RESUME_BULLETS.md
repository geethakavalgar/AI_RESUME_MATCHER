# Resume Bullets

## AI Resume Matcher
- Built an AI-powered resume matching application using Python, Sentence Transformers, scikit-learn, and Streamlit to compare resumes against job descriptions with semantic similarity scoring.
- Implemented PDF resume parsing and keyword-based skill-gap analysis to identify missing capabilities and improve resume-job alignment insights.
- Designed an interactive UI that surfaced match scores, tracked skill coverage, and generated targeted improvement suggestions for end users.

## RAG PDF Chatbot
- Developed a retrieval-augmented generation (RAG) chatbot that ingests PDF documents, creates embeddings, indexes content in FAISS, and answers user questions with source-aware retrieval.
- Implemented document chunking, embedding-based semantic search, and grounded response generation to reduce hallucinations and improve answer relevance.
- Built a Streamlit-based interface for multi-document upload, question answering, and source-page display to demonstrate an end-to-end LLM application workflow.
