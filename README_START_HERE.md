# AI Portfolio Pack

This package contains two starter AI engineering projects you can push to GitHub:

1. **ai-resume-matcher**
2. **rag-pdf-chatbot**

It also includes a deployment guide and ready-to-use resume bullets.

## Suggested Next Steps
1. Unzip the package.
2. Create two GitHub repositories.
3. Push one project into each repository.
4. Run each project locally.
5. Add screenshots to each README after testing.
